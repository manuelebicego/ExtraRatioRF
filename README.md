# ExtraRatioRF
Matlab code for computing the ExtraRatioRF distance -- See the file Demo.m

Paper "ExtraRatioRF: an extremely randomized Random Forest distance", submitted to ECML-PKDD 26

NOTE: 
The code requires the Random Forest Clustering library, available from the authors of the competitor RatioRF distance [1] at their web page: 

https://profs.scienze.univr.it/~bicego/code.html

We provide here the version downloaded at the moment of the submission

[1] M. Bicego, F. Cicalese, A. Mensi: "RatioRF: a novel measure for Random Forest clustering based on the Tversky's Ratio model", IEEE Transactions on Knowledge and Data Engineering, vol. 35, no. 1, pp. 830--841, (2023)
