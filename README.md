# ExtraRatioRF
Matlab code for computing the ExtraRatioRF distance

Paper "ExtraRatioRF: an extremely randomized Random Forest distance", submitted to ECML-PKDD 26

% See Demo.m

The code requires the Random Forest Clustering library, available from the authors of RatioRF at their web page: 

https://profs.scienze.univr.it/~bicego/code.html

We provide here the version downloaded at the moment of the submission
